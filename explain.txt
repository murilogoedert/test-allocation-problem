D - set de carteiras
T - set de testes
Ytu - Matriz de similaridade entre o teste 't' vs o teste 'u' pertencentes a T
E - sub set de D que estão proximos o suficiente para serem considerados.
Pde - Matriz de distancia entre a carteira 'd' vs a carteira  'e' persentecentes a E
S - numero de estudantes que vão fazer o teste sendo <= a |D|
F - numero de carteiras vazias sendo igual a |D| - S
T+ - Set de testes T mais um teste '0' que será definido a carteiras vazias

Função objetivo:
Somatório de distancias de E
